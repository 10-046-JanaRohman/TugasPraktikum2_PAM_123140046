## Fitur Aplikasi
Aplikasi **News Feed Simulator** merupakan simulasi tampilan berita yang akan terus diperbarui secara otomatis.

1. Daftar berita akan bertambah setiap 2 detik secara otomatis
2. Tersedia fitur penyaringan berita berdasarkan kategori tertentu
3. Informasi berita ditampilkan dengan susunan yang terstruktur (judul dan kategori)
4. Terdapat fitur penghitung jumlah baca (Read Count)
5. Ketika berita dipilih, aplikasi akan menampilkan halaman detail yang dimuat secara asynchronous (simulasi)

## Panduan Menjalankan Aplikasi
1. Unduh project ini terlebih dahulu
2. Ekstrak file ZIP yang telah diunduh
3. Buka project menggunakan Android Studio
4. Tunggu hingga proses Gradle Sync selesai sepenuhnya
5. Jalankan aplikasi dengan menekan tombol **Run**  
   Kemudian pilih emulator atau perangkat Android yang tersedia
6. Setelah aplikasi terbuka, daftar berita akan tampil dan bertambah secara otomatis
